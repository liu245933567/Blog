## 防火墙

1、关闭防火墙：sudo systemctl stop firewalld.service

```bash
sudo systemctl stop firewalld.service
```

2、关闭开机启动：sudo systemctl disable firewalld.service

```bash
sudo systemctl disable firewalld.service
```
