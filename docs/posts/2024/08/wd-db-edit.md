## 修改了等级

92级

```txt
9: 92,
44: 364,
45: 92,
```

93级别

```txt
9: 93,
44: 368,
45: 93,
```

94级别

```txt
9: 94,
44: 372,
45: 94,
```

得出结论 44 = 4 \* (9 - 1)

## 修改了性别

女

```txt
56: 2,
```

男

```txt
56: 1,
```

得出结论 1: 男 2: 女

## 修改门派

金

```txt
19: 1,
```

木

```txt
19: 2,
```

水

```txt
19: 3,
```

得出结论 1: 金 2: 木 3: 水 4: 火 5: 土

## 修改了道行

899年

```txt
49: 323640,
```

1000年

```txt
49: 360000,
```

得出结论 一年 = 360天 该值存储的是天数
