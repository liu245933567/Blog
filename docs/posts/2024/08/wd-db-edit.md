## 修改了等级

92级

```txt
9: 92,
44: 364,
45: 92,
```

93级别

```txt
9: 93,
44: 368,
45: 93,
```

94级别

```txt
9: 94,
44: 372,
45: 94,
```

得出结论 44 = 4 \* (9 - 1)

## 修改了性别

女

```txt
56: 2,
```

男

```txt
56: 1,
```

得出结论 1: 男 2: 女

